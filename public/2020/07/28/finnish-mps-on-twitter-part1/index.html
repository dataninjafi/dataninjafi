Here is my take on analysis of tweets of Finnish MPs. Goals for this project:

  • Automate hourly collection of tweet data produced by MPs.
  • Describe how Finnish MPs tweet based on their background variables. Atleast by party, age, gender and MP seniority.
  • Sentiment analysis of the language used in the tweets.
  • Update constantly open source raw data and publish weekly a report on language used.

Of course all this done when 👶 allows me to code.

Finnish MPs on Twitter

First we can get listing of current MPs and their backgrounds on wikipedia https://fi.wikipedia.org/wiki/Luettelo_vaalikauden_2019%E2%80%932023_kansanedustajista. Let’s use rvest-package

library(rvest)
## Loading required package: xml2
url <- "https://fi.wikipedia.org/wiki/Luettelo_vaalikauden_2019%E2%80%932023_kansanedustajista"

wikipedia <- read_html(url)

wikipedia %>% 
  html_node("body #content #bodyContent #mw-content-text")
